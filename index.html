Dispatching rules applied:
1. Gap < 30 min between legs → same driver waits and returns the member.
2. Gap 1–2 hrs between legs (or at least 1 hr free before next pickup) → same driver keeps the return.
3. Gap > 2 hrs or Will Call → return moves to a less-loaded driver only if the original driver is busy with another pickup; otherwise the original driver keeps it.
4. Cost and trip count balanced as evenly as possible across all drivers.
5. Last trip of the day assigned to the same driver as the trip before it.
