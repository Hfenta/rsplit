ROUTESPLIT v2.0 — PWA Install Guide
=====================================
WHAT'S NEW IN v2.0:
  - Working Excel cell colors (built-in OpenXML writer)
  - Unassigned sheet exported in Excel + Word
  - Phone numbers as clickable tel: links
  - 60 min gap between outbounds, 45 min outbound/return gap
  - Pickup address used as name when no member name in file
  - Schedule date in export filenames
  - Works fully offline after first load

DEPLOY (2 min, free):
  1. Go to https://app.netlify.com/drop
  2. Drag the "RouteSplit_PWA" folder onto the page
  3. Get your URL instantly

GITHUB PAGES:
  1. New repo → "routesplit" → Public
  2. Upload all files from this folder (not the folder itself)
  3. Settings → Pages → Branch: main / root → Save

INSTALL ON PHONE:
  iPhone  → Safari → Share → Add to Home Screen
  Android → Chrome → Install banner or Menu → Add to Home Screen

INSTALL ON LAPTOP:
  Chrome / Edge → click install icon in address bar

After the first online load, the app and all exports work
fully offline — no internet connection needed.
